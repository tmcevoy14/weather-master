If you're like most people and don't like being stranded in the rain, turn this notification on to be alerted as soon as a shower starts! Simply press press the button to turn the notification on. If you have a specific amount of rain that you want to be notified about, you can enter an amount of rain in mm into the input box. If the hourly rain goes above that amount, you will be alerted.
Wind notifications will alert you as soon as the wind goes above a given speed. Even if you're not a wind surfer it's always good to be aware! Simply press the button to turn the notification on. If you have a specific wind speed that you want to be notified about, you can the value into the input box. You will be alerted as soon as the wind goes above that value!
No need to worry about being cold anymore, the temperature notifications will alert you as soon as the temperature drops below or rises above a specific temperature. Because all people have a different sense of warm and cold, make sure to enter your own temperature value to be alerted when you should bring a jacket with you! Simply enter a number into the input box and press the button to turn these notifications on.

If you want to be notified when the current temperature goes above a certain level then input your value in the High Temp box.

If you want to be notified when the current temperature goes below a certain level then input your value in the Low Temp box.
Barometric Pressure readings can be used to help you predict the future weather! Dropping pressure can indicate a clearing sky, and a rising pressure can indicate rain. Turn this notification on to be alerted of significant drops in pressure!

If you want a more specific notification you can enter your own values into both input boxes.

Time Range: represents how many hours ago that you want to compare the pressure to.

Drop Range: represents how much of a drop you want to be notified about.

So, if you input 3 into Time Range and 5 into Drop Range, you will be notified if the pressure has dropped by 5 mBar in the past 3 hours!
